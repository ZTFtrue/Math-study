# 幂运算(指数运算)

$n^m$ n为底数，m为指数。n的m次幂、n的m次方
$0^0$ 没有正式的定义，部分惯例为1
幂不符合结合律和交换律
$a^0=1$
$n^{a}\cdot n^{b}=n^{(a+b)}$
$n^{a}\div n^{b}=n^{(a-b)}$
$(a^m)^n=a^ { m\cdot n} $
$a^{n}\cdot b^{n}=(a\cdot b)^{n}$
$a^{n}\div b^{n}=(a\div b)^{n}$
$a^0 =1=a^{1-1}=\frac{a^{1}}{a^{1}}$
$a^{(-k)}=\frac{1}{a^{k}}=a^{(0-k)}=\frac{a^{0}}{a^{k}}=\frac{1}{a^{k}}$
$a^{\frac{m}{n}}=\sqrt[n]{a^{m}}(n\neq 0)$
